Tentang Saya

Halo! Saya Iqbal Guntur Bismoko, seorang mahasiswa aktif di Program Studi Pendidikan Teknik Informatika dan Komputer (PTIK) Universitas Negeri Semarang. Perjalanan saya di dunia teknologi dimulai dari SMK dengan jurusan Teknik Komputer dan Jaringan (TKJ), di mana saya pertama kali tertarik dengan dunia Linux dan sistem operasi.

Saat ini, saya fokus mengembangkan kemampuan di bidang pemrograman, khususnya dalam pengembangan aplikasi web dan sistem. Saya memiliki ketertarikan khusus pada ekosistem Linux, scripting dengan Bash, dan pemrograman dengan Python. Bagi saya, teknologi bukan hanya tentang menulis kode, tetapi tentang menciptakan solusi yang memberikan dampak positif.

Melalui berbagai project akademik dan kolaborasi dengan teman-teman, saya terus belajar dan mengasah kemampuan technical maupun soft skill seperti problem solving dan teamwork.
